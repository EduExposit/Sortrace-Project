Cargar datos

Introducir datos por teclado

Seleccione desde el menú la opción "Archivo" para mostrar las posibles acciones.

Pulsamos sobre el botón de "Introducir Datos" de la barra de herramientas (El primer botón).

A continuación, se mostrará una ventana en la que deberá introducir el tamaño deseado del vector y pulsar Aceptar.

Para cancelar la accion y volver al inicio pulsar en el boton cancelar Cancelar.

Para finalizar, deberá introducir los elementos del vector en la nueva pantalla y pulsar Aceptar.

Para cancelar la accion y volver al inicio pulsar en el boton cancelar Cancelar.

Generar datos aleatorios

Seleccione desde el menú la opción "Archivo" para mostrar las posibles acciones.

Pulsamos sobre el botón de "Generar Datos Aleatorios" de la barra de herramientas (El segundo botón).

A continuación, se mostrará una ventana en la que deberá introducir el tamaño deseado del vector aleatorio, el tipo de vector aleatorio a generar (vector aleatorio con elementos desordenados, en orden creciente o decreciente), y de manera opcional puede determinar los límites inferior y superior de los elementos del vector aleatorio.

Nota: En caso de no introducir valores específicos para los límites, se generará por defecto un vector aleatorio con elementos comprendidos entre 0 y 99 ambos incluidos. Para finalizar la generación de datos aleatorios debe pulsar Aceptar.

Cargar datos de fichero

Seleccione desde el menú la opción "Archivo" para mostrar las posibles acciones.

Pulsamos sobre el botón de "Cargar Datos..." de la barra de herramientas (El tercer botón).

A continuación, seleccione el fichero que desea cargar y pulse Abrir.

Nota: El fichero de entrada del vector debe ser un fichero con formato txt y deberá contener una lista de números separados por un espacio en blanco, donde el primer número representa la longitud del vector y el resto de números son cada uno de los elementos que contiene.
Los valores mínimo y máximo permitidos para los elementos del vector son -999 y 999 respectivamente.
