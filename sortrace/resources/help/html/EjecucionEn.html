Execution

Pause

Stops continuous execution of the algorithm. It is equivalent to selecting the pause icon from the icon bar.

Go back to the beginning

Go back to the beginning of the algorithm's execution. It is equivalent to select the triple arrow icon to the left from the icon bar.

Go back to the beginning of the iteration

Go back to the beginning of the last executed iteration of the algorithm. It is equivalent to selecting the double arrow icon towards the left from theicon bar.

Go back one step

Go back one step in the execution of the algorithm. It is equivalent to selecting the left arrow icon (ending in a vertical bar) from the icon bar.

Backward continuous

Go back the execution of the algorithm continuously step by step. It is equivalent to selecting the arrow icon to the left from the icon bar.

Advance continuous

It advances the execution of the algorithm continuously step by step. It is equivalent to selecting the arrow icon to the right of the icon bar.

Take one step forward

Go one step further in the execution of the algorithm. It is equivalent to selecting the icon of the right arrow (ending in a vertical bar) from the icon bar.

Advance to the end of the iteration

Advances to the end of the current iteration of the algorithm. Is equivalent to select the double arrow icon to the right from the icon bar.

Advance to the end

Advances to the end of the algorithm execution. It is equivalent to select the triple arrow icon to the right from the icon bar.

Exit execution

It exit the execution of the algorithm, returning to its initial state. It is equivalent to selecting the cross icon on the icon bar.
