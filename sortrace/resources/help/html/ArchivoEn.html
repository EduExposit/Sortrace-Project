File

Introduce data

Shows a dialog in which the size of the vector to order is requested, later a new dialogue asks to introduce the elements of the vector. It is equivalent to selecting the computer icon from the icon bar.

Generate random data

It shows a dialog in which you are asked to enter the size of the vector, the type of random generation and, optionally, the lower and higher limits than the elements of the random vector to be generated. It is equivalent to selecting the dice icon from the icon bar.

Load data

Displays a dialog asking to select a text file with vector data. It is equivalent to selecting the icon of the folder from the icon bar.

Modify data

Displays a dialog in which it is possible to modify each of the vector elements. It is equivalent to selecting the notebook icon from the icon bar.

Save file

Shows a dialog in which you are asked to select a text file to save the vector. It is equivalent to selecting the disk icon from the icon bar.

Export Image

Displays a dialog asking to select a file with the format jpg or png in which you want to save the current view of the cumulative trace. It is equivalent to selecting the icon in the image from the icon bar.

Export Image Sequence

Displays a dialog in which you are asked to select a directory, you want to save the image sequence of the cumulative traces that make up the full execution of the algorithm. It is equivalent to selecting the image mosaic icon from the icon bar.

Export animation

Displays a dialog asking to select a file with the format gif, in which you want to save the animation of the cumulative traces that make up the complete execution of the algorithm. It is equivalent to selecting the video tape icon from theicon bar.

Exit

Displays a dialog asking for confirmation to close the application.
