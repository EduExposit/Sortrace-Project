Execute algorithm

Advance continuous

Select from the menu the option "Execution" to display the possible actions.

Click on the "Continuous Advance" button on the toolbar (The sixth button).

By activating this execution mode the algorithm will be executed continuously until the execution is finished or until it is stopped by pressing the option Abandon execution or Pause.
This option will appear enabled at all times during the execution of the algorithm, except if the algorithm is in continuous execution mode or the algorithm is in the final step.

Avanzar un paso

Select from the menu the option "Execution" to display the possible actions.

Click on the button "Advance Step" of the toolbar (The seventh button)

Activating this execution mode will execute one step of the algorithm.
This option will appear enabled at all times during the execution of the algorithm, except if the algorithm is in continuous execution mode or the algorithm is in the final step.

Advance to the end of the iteration

Select from the menu the option "Execution" to display the possible actions.

Click on the button "Advance to the end of the iteration" of the toolbar (The eighth button).

Activating this execution mode will execute an entire iteration of the algorithm.
This option will appear enabled at all times during the execution of the algorithm, except if the algorithm is in continuous execution mode or the algorithm is in the final step.

Avanzar al final

Select from the menu the option "Execution" to display the possible actions.

Click on the button "Advance to the end" of the toolbar (The ninth button).

This option will appear enabled at all times during the execution of the algorithm, except in the case that the algorithm is in continuous execution mode or the algorithm is in the final step, and runs the algorithm to the end.

Pause

Select from the menu the option "Execution" to display the possible actions.

Click on the "Pause" button on the toolbar (The first button)

This option will appear enabled only when the algorithm is in continuous execution mode, and pauses the execution of the algorithm.

Exit execution

Select from the menu the option "Execution" to display the possible actions.

Click on the button "Exit Execution" of the toolbar (The tenth button).

This option will appear enabled when the algorithm is in a state intermediate execution, either in continuous execution mode or stepping mode, and it will abandon the execution of the algorithm, returning to the initial state of the same.

Go back to the beginning

Select from the menu the option "Execution" to display the possible actions.

Click on the button "Back to the beginning" of the toolbar (The second button).

This option will appear enabled when the algorithm is in an intermediate state execution time or the algorithm has been fully executed, and allows return to the initial state of execution.

Retroceder al comienzo de la iteración

Select from the menu the option "Execution" to display the possible actions.

Click on the button "Go back to the beginning of the iteration" of the toolbar (The third button).

This option will appear enabled at all times during the execution of the algorithm, except in the case that the algorithm is in continuous execution mode or the algorithm is in the first step, and goes back to the beginning of the last iteration executed.

Go back one step

Select from the menu the option "Execution" to display the possible actions.

Click on the "Back one step" button on the toolbar (The fourth button).

This option will appear enabled at all times during the execution of the algorithm, except in the case that the algorithm is in continuous execution mode or the algorithm is in the first step and takes one step back in the execution of the algorithm.

Backward continuous

Select from the menu the option "Execution" to display the possible actions.

Click on the button "Continuous backward" of the toolbar (The fifth button).

By activating this execution mode the algorithm will be executed continuously until reach the initial state of the algorithm or until it is stopped by pressing the option Exit execution or Pause. This option will appear enabled at all times algorithm execution, except if the algorithm is in execution mode continuous or the algorithm is in the first step.
