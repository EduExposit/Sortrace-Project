Load data

Enter data by keyboard

Select from the menu the option "File" to show the possible actions.

Click on the button "Enter Data" from the toolbar (The first button).

Next, a window will appear in which you must enter the size desired vector and press Accept.

To cancel the action and go back to the beginning, press cancel.

Finally, you must enter the elements of the vector in the new screen and press Accept.

To cancel the action and go back to the beginning, press cancel.

Generate random data

Select from the menu the option "File" to show the possible actions.

Click on the button "Generate Random Data" of the toolbar (The second button).

Will appear a dialogue that will ask us if we want to customize the data generation or we want the system to generate them automatically without specifying parameters.

Then it will be displayed a window where you should introduce the desired size of the random vector, the type of random vector to generate (random vector with disordered elements, in increasing or decreasing order), and optionally you can determine the lower and higher limits of the random vector elements.

Note: In case of not entering specific values for the limits, will be generated by default a random vector with elements between -999 and 999 both included. To finish the random data generation you must press accept.

Load data from file

Select from the menu the option "File" to show the possible actions.

Click on the button "Load Data ..." from the toolbar (The third button).

Then select the file you want to upload and press Open.

Note:The vector input file must be a file with format txt y deberá contener una lista de números separados by whitespace, the first number represents the length vector and the rest of the numbers are each of the elements it contains.
The minimum and maximum values allowed for the elements of the vector they are -999 and 999 respectively.
