Settings

Visualization

Displays a dialog where the user is allowed to change the colors used for animation between vector elements. It is equivalent to selecting the visualization icon from the icon bar.

Language

Displays a dialog in which the user is allowed to select the language of the application, between the two possible, Spanish and English. It is equivalent to selecting the language icon from theicon bar.

Animation

Displays a dialog in which the user is allowed to set the speed continuous execution of the algorithm. It is equivalent to selecting the animation icon from theicon bar.
